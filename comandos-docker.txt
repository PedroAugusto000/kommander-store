Criando uma network: docker network create kommanderstore-network

Criando container e rodando: 
docker run -d --name postgres --network kommanderstore-network -e POSTGRES_USER=admin -e POSTGRES_PASSWORD=admin -e POSTGRES_DB=kommanderstore -p 5432:5432 postgres

docker run -d --name pgadmin4 --network kommanderstore-network -e PGADMIN_DEFAULT_EMAIL=admin@admin.com -e PGADMIN_DEFAULT_PASSWORD=admin -p 5050:80 dpage/pgadmin4

docker run -d --name=dev-consul -e CONSUL_BIND_INTERFACE=eth0 -p 8500:8500 hashicorp/consul
